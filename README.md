# Cross-Phoneme Generalisation in Dimension-Based Statistical Learning

Analysis code and data for:

Segger-Staveley, S., He, M., & Steffman, J. (accepted - in pre-print). Cross-phoneme 
generalisation in dimension-based statistical learning for stop voicing: probing 
subject design and word frame effects.

Originally hosted on [OSF](https://osf.io/w2dzt/).

## My Contribution

I am first author on this paper. Along with my collaborators, I designed and ran the experiments, conducted 
the statistical analysis, and led the writing up of results.

## Methods

- **Language:** R
- **Statistical analysis:** Mixed-effects regression (lme4), Bayesian inference (brms)
- **Data processing:** tidyverse, dplyr
- **Visualisation:** ggplot2
